# Cameracal Services – Sensor Health Check V1.4

Static web app for JPEG-based sensor dust verification.

## V1.4 changes
- Larger Cameracal Services logo in the app header.
- Larger Cameracal Services logo in the printable PDF report.
- JPEG-only upload wording and JPEG-only file validation.
- Updated test-image guidance: F16 / F22, white background or blue sky.
- Paid PDF report now includes an Aperture Visibility Estimate covering f/8, f/11, f/16 and f/22.
- UI wording notes that RAW files should be exported to JPEG for V1; RAW can be added later as an Advanced / Pro phase.
- Footer uses professional SVG-style service icons.
